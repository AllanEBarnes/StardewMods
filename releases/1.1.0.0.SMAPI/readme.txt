-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
AllProfessions v1.1.0.0.SMAPI
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Upon leveling up, will add all possible professions to that character instead of just the one chosen.
By cantorsdust.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
INSTALLATION
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

The zip contains one folder, AllProfessions, with two files AllProfessions.dll and manifest.json. The folder must be placed in %appdata%\StardewValley\Mods.

Thus, the total path for both of the files required for this mod to function are:
%appdata%\StardewValley\Mods\AllProfessions\AllProfessions.dll

AND

%appdata%\StardewValley\Mods\AllProfessions\manifest.json


REQUIRES SMAPI 0.39+ to be installed!
https://github.com/ClxS/SMAPI/releases


-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
USAGE
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Run StardewModdingAPI.exe in your main Stardew Valley folder. This will load the mods and then start the game.
Upon leveling up, all professions will be added to your character instead of just your chosen one.  For example, both Rancher and Tiller instead of just Rancher.
Characters who already have professions will have any profession they are eligible for added upon the next location change.
